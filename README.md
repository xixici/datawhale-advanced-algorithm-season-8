# Advanced-algorithm-combing

**课程设计**：黑桃，刘广月，于鸿飞

**组队学习说明**：通过查阅相关文献，对机器学习算法进行梳理

**任务路线**：RF--->GBDT--->XGB--->LightGBM

**组队学习周期**：10天

**定位人群：有概率论、矩阵运算、求导、泰勒展开等基础数学知识；**

**难度系数**：中

**每个任务完成大概所需时间：2-3h**
